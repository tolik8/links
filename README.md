Links - сервис хранения своих ссылок.

Интерфейс на трех языках: en, ru, ua
Ссылки можна хранить в группах (группы древоподобные).
Возможность видеть ссылки друзей.
Возможность видеть ссылки в разном виде.
Возможность поиска ссылок по имени, описанию и по тегам.

Фреймфорк: Laravel 5.8.*

Цель проекта:
- личное использование
- опыт разработки на Laravel
- для портфолио
- возможность заработать - врятли (но донат прикрутить можно)
  но ни в коем случае не рекламу, реклама убьет проект
